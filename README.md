# 🎌 Lexi Subs

### Anime subtitle overlay & Japanese study companion

**Watch anime. Click words. Learn Japanese. Continue claiming you're "just watching one more episode."**

Lexi Subs transforms subtitles into an interactive learning experience directly in your browser.

No copy-pasting. No tab juggling. No interrupting immersion.

## ✨ Features

### 🎌 Universal subtitle overlay

Bring subtitles almost anywhere.

✅ `.srt`, `.vtt`, `.ass`

✅ Streaming sites & local videos

✅ Customize:

* Font
* Size
* Position
* Colors
* Outline

---

### 🔍 Built-in subtitle search

Search and load subtitles directly inside the extension.

Supported sources:

* Jimaku
* OpenSubtitles

Because this:

```txt
Anime → Google → Random site
      → Ads → Wrong file
      → Suffering
```

is a side quest nobody asked for.

---

### 📖 Instant Japanese lookup

Hover or click words to instantly see:

* Kana reading
* Definitions
* JLPT level
* Common-word information

Powered by Jisho.

---

### 🧠 Study mode

Turn subtitles into mini vocabulary lessons directly below the video.

Passive watching suddenly becomes:

```txt
Watching anime...
        ↓
Oops, accidentally studied Japanese
```

---

### ⏱️ Precision subtitle sync

Subtitle timing off by approximately 3 business days?

Quick adjustments:

`±10ms`

`±100ms`

`±1000ms`

---

### 🔁 Auto next-episode support

Continue watching without repeatedly setting everything up again.
